# PFC_ConversorBoost

Aplicação Streamlit para conversão e análise de dados de PFC (Projeto Final de Curso).

## Descrição

Este projeto fornece uma interface web intuitiva para conversão e processamento de dados.

## Instalação

```bash
pip install -r requirements.txt
```

## Como executar

```bash
streamlit run app.py
```

## Estrutura do projeto

- `app.py` - Arquivo principal da aplicação Streamlit
- `requirements.txt` - Dependências do projeto
- `src/` - Módulos e utilitários

## Tecnologias

- Streamlit
- Python 3.x
